# To build

## Dev version

`docker build -f Dockerfile.dev -t website .`
docker run -p 3000:3000 website

## Prod version

```
docker build -f Dockerfile.prod -t website .
docker run -p 3000:3000 website
```

Though prod is usually run from `rome` docker compose
